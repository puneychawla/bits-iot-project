City-based Air Pollution Monitoring System



About this project

This project makes use of the LoRa technology for the communication and has been built over it. The module is based over a drone which powers the unit and helps collect data through the MQ 135 Air Quality sensor on board. The data collected in this process is relayed over the cloud by the LoRa module and is processed to be displayed on this website.


Submission

This project was submitted to Prof Vinay Chamola, as a partial fulfilment of the course EEE F411 - Internet of Things.

This website was made for Internet of Things(EEE F411) Course Project and we do not hold any liability for accuracy of the data as this project is in an experimental phase.
